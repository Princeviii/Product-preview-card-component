## Overview

### The Challenge

Build a responsive product preview card component with mobile and desktop layouts, using HTML, CSS, and Flexbox. The goal was to create a visually appealing card that matches modern e-commerce design patterns and includes a working “Add to Cart” button.

## My Process

### Built With

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- CSS Grid
- Mobile-first workflow
- Google Fonts (Montserrat, Fraunces, Figtree)

### What I Learned

- How to implement responsive images using the `<picture>` element.
- Proper semantic HTML structure using `<article>` and `<section>`.
- Using Flexbox for both column and row layouts.
- Handling overflow and spacing for clean layouts.
- Creating a visually appealing “Add to Cart” button with hover effects.
